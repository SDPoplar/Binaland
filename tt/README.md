# Binaland
[Github - Binaland](https://github.com/SDPoplar/Binaland)

> tt = Tiny tools

## tplmaker
save template file into
```
const char* ...
```
then can be used in binaland

```
tplmaker
```
or
```
tplmaker --help
```
can get the usage

### Complex tplmaker
```
g++ -std=c++11 -lboost_regex main.cpp -o tplmaker
```
